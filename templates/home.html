Notes

I tried my best to make this flexible enough to read in any Trackman dataset without

spending too much time worrying about scalability. If I were to spend more time on this

project, I would like to add functionality that would allow the user to upload a csv file

to the server, and then if that data was Trackman data, read it into a database instead of

reading directly from csv files and make the tables sortable and downloadable.

That being said I am very proud of this project and I hope you enjoy this as much as I did making it!
